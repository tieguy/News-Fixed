## Fun Facts About the Code

1. **Why 4 days?** The newspaper comes out Monday-Thursday (weekdays before Friday). Each day has a different theme!

2. **Why QR codes?** So you can learn more! Scan them with a phone to read the original news story.

3. **Why does Claude help?** News for adults uses big words and boring sentences. Claude rewrites everything so it's interesting for you!

4. **What's a PDF?** It stands for "Portable Document Format" - it's a file that looks the same no matter what computer opens it. Perfect for printing!

5. **What's caching?** Just like Scratch projects can save your progress, the code saves stories it already downloaded so it doesn't have to get them again. Saves time!

## If This Were a Scratch Project...

It would have:
- **5 main sprites** (Fetcher, Sorter, Writer, Maker, Director)
- **3 helper sprites** (XKCD, QR Code, Sports)
- **About 15 broadcasts** (messages between sprites)
- **Many lists** (to store all the stories)
- **Cloud variables** (to talk to websites)

The biggest difference from Scratch? Instead of blocks you drag and drop, this uses **Python code** - which is text you type. But the IDEAS are the same: sprites, messages, lists, and making things happen in order!
